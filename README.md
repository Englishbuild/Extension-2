# Video Downloader Plus - Free & Open Source

A completely free and open source browser extension for downloading videos from various websites.

## 🎉 Now Completely Free!

This extension has been converted to be **100% free and open source**! No more premium restrictions, login requirements, or server dependencies.

## ✨ Features

- **Download videos** from most websites in various formats (MP4, MOV, FLV, WebM, 3GP, OGG, M4A, MP3, WAV, BIN)
- **All formats available** - including MP3 which was previously premium-only
- **No login required** - works completely offline
- **No server dependencies** - all processing happens locally
- **Save videos for later** - local bookmarking system
- **Chromecast support** - stream videos to your TV
- **Customizable settings** - choose file types and minimum video sizes
- **Clean, modern interface** - easy to use popup and options pages

## 🚀 Installation

### From Source (Recommended)
1. Clone or download this repository
2. Open Chrome/Edge and go to `chrome://extensions/` or `edge://extensions/`
3. Enable "Developer mode" in the top right
4. Click "Load unpacked" and select the extension folder
5. The extension icon should appear in your browser toolbar

### Building
No build process required! This is a pure JavaScript extension that runs directly in the browser.

## 📖 Usage

1. **Navigate** to any website with videos
2. **Click** the extension icon in your toolbar
3. **Download** any detected videos directly
4. **Configure** settings by clicking the gear icon in the popup

## 🔧 Configuration

Access settings through:
- Click the extension icon → Settings (gear icon)
- Or go to `chrome://extensions/` → Extension details → Extension options

### Available Settings:
- **Video Types**: Choose which formats to detect (MP4, MOV, FLV, WebM, 3GP, OGG, M4A, MP3, WAV, BIN)
- **Minimum File Size**: Filter out small files (100KB, 1MB, 2MB)
- **Chromecast**: Enable/disable casting functionality
